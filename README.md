## 🙋‍♀️ About Me

I'm a software engineering student who believes that **great code is not just about making things work — it's about making them work beautifully**.

My journey started with curiosity: *Why does software break? How do programs think? What makes one solution better than another?* These questions pulled me deeper into the world of computer science, and I haven't looked back since.

Right now, I'm focused on building a rock-solid foundation. That means getting comfortable with **Java**, understanding how **Object-Oriented Programming** shapes the way we model the real world in code, and learning how to write software that others can actually read and maintain. I care a lot about clean code — not as a rule to follow, but as a mindset to develop.

Outside the IDE, I enjoy breaking down complex problems into smaller pieces and figuring out the most logical path forward. I believe **algorithmic thinking** is one of the most transferable skills a developer can have, and I'm actively working to sharpen it every day.

I use this space to document my progress, share my projects, and hold myself accountable. Every repository here is a snapshot of where I was and how far I've come. Some projects are small, some are messy — but all of them represent genuine effort and learning.

I'm just getting started, and I'm excited about where this path leads. 🚀
